GitHub Tutorial for Mac

1.   Install Git

To install Git, go to the google search bar and type in "install git." One of the first looks that should pop up is a tutorial from github's website. Click on said site. After the page has loaded, scroll down until you see the heading "Install on Mac." There will be a link to install "macOS Git Installer." Click on said link to install Git on your device.

2.   Install VS Code

To install VS Code, go to their officially website and click on the "Download" button in the top right corner. On this page click on the "Mac" button to install VS Code.

3.   Enable Github on VS Code

- Go to File > Preferences
- Go to Settings
- Type Git: Enabled in the search bar
- Make sure that the box is ticked

4.   Create Account or Sign in to Github

5.   Configure Github in VS Code

Open up Terminal and type "$ git config --global user.name "Your Name" and the click enter. After that type type in "$ git config --global user.email "youremail@github.com" and then click enter.

6.   Verify Git is linked to Github

When your details are configured in Git, you can type in "$ git config --global --list" in the Terminal and click enter. Your previously typed email and username should pop up. You are connected to Git through VS Code!





Cloning Repository

1.   GitHub Repository

- Go to Github and log in to account
- Go to your repository and click on the "Code" button
- Copy the HTTPS url

2.   VS Code

- Go to VS Code
- Open command palette and press "cmd + shift + P"
- Type "Git:Clone"
- paste HTTPS url

3.   Select Repository Location

- You will be prompted to you choose an existing folder or create a folder to place this file
- A window pop-up will appear asking to open in new window or existing
- Click on desired option and you repository is cloned
